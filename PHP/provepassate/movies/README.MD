# Richieste (NB: sono R e C in **CR**UD)

2.  Realizzare un form HTML che permetta l'inserimento di un nuovo record nella tabella **movies**, verificando che siano visibili i nuovi inserimenti.

# Facoltative (NB: sono U e D in CR**UD**)
3.  Nella lista di record ricevuta dal server come codice HTML con (1), si renda, per ciascun record, l'attributo *isbn* un link che cliccato porta a un form di aggiornamento (modifica) dei record di provenienza.
4.  Aggiungere, in quest'ultimo form di aggiornamento, un bottone per l'eliminazione dei record dalla tabella **movies**.
